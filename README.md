## Instalation

```sh
git clone https://github.com/JanKantyPawelski/react-boilerplate.git
cd react-boilerplate
npm install
```

## Usage

Production build:

```sh
npm run build
```

Development build::

```sh
npm run dev
```

Run tests:
```sh
npm run test
```

Lint code:

```sh
npm run lint
```

* `port` - port for development build
